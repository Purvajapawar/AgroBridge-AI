# AgroBridge AI

## Problem
Farmers struggle to get fair prices and market access.

## Solution
AgroBridge AI connects farmers with traders using AI price prediction.

## Features
- Crop listing
- Trader marketplace
- AI price prediction
- Market analytics

## Tech Stack
Frontend: React / Expo
Backend: Node.js / Flask
AI: Python, Scikit-learn
